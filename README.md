# Finalproject - Radio Planet
Setup
Fork & Clone
Run npm install to install dependencies
Run npm start to start the server
Open http://localhost:3000

#Dependencies

    @craco/craco 5.2.1
    cesium 1.57.0
    craco-cesium 1.0.0
    react 16.8.6
    react-audio-spectrum 0.0.9
    react-dom 16.8.6
    react-hot-loader 4.8.8
    react-player 1.11.0
    react-scripts 3.0.1
    resium 1.4.1
